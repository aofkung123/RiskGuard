# RiskGuard Platform

RiskGuard is a construction project management platform designed to mitigate cost and time risks.

## Features
- **For Contractors**: Project management, Cost calculation, Portfolio showcase, CPI/SPI tracking.
- **For Employers**: Contractor matching, Chat & Blueprint sharing, Project monitoring.
- **For Data Analysts**: ETL pipelines, Scrapers for material prices, Risk prediction models.

## Tech Stack
- **Frontend**: Next.js, Tailwind CSS, TypeScript.
- **Backend**: FastAPI, PostgreSQL, SQLAlchemy.
- **Analytics**: Python (Scikit-learn, Pandas).

## Getting Started

### Backend
1. Go to `backend/`
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python run.py`

### Frontend
1. Go to `frontend/`
2. Install dependencies: `npm install`
3. Run: `npm run dev`
