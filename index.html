Early Warning ตรวจจับสัญญานเตือนภัยก่อนสายเกินแก้

ตัวชี้วัดความมั่นคงของโครงการ (EVM)

เราใช้หลักการ Earned Value Management (EVM) เพื่อคำนวณความเสี่ยงของโครงการอย่างมีมาตรฐาน

CPI — Cost Performance Index

ดัชนีประสิทธิภาพต้นทุน

สูตรคำนวณ

CPI = EV / AC

≥ 1.0 → อยู่ในงบประมาณ
< 0.95 → เริ่มมีความเสี่ยง
< 0.85 → วิกฤตงบเกิน

SPI — Schedule Performance Index

ดัชนีประสิทธิภาพเวลา

สูตรคำนวณ

SPI = EV / PV

≥ 1.0 → การทำงานตรงตามแผน
< 0.95 → เริ่มล่าช้าสะสม
< 0.85 → วิกฤตล่าช้ามาก

Variance — ค่าความเบี่ยงเบน

ส่วนต่างระหว่างแผนกับจริง

สูตรคำนวณ

Variance = Actual − Planned

≤ ±10% → ปกติไม่มีปัญหา
10–15% → ควรได้รับการเฝ้าระวัง
> 15% → วิกฤตต้องแก้ไขด่วน
